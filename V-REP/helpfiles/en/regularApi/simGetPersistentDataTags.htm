Regular API function

simGetPersistentDataTags / sim.getPersistentDataTags

Description Retrieves all persistent data block tags or names. See also sim.getMatchingPersistentDataTags and sim.persistentDataRead
C synopsis simChar* simGetPersistentDataTags(simInt* tagCount)
C parameters
tagCount: the number of retrieved data block tags
C return value
NULL if operation was not successful or no data block exists. Otherwise the data block tags, separated by the '\0' char. In that case the returned buffer should be released with simReleaseBuffer
Lua synopsis
table tags=sim.getPersistentDataTags()
Lua parameters
-
Lua return values
tags: a table containing the retrieved data block tags.
