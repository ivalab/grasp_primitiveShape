Regular API function

simStopSimulation / sim.stopSimulation

Description Requests a stop of the running simulation. See also sim.startSimulation, sim.pauseSimulation and sim.getSimulationState. See also the simulation state diagram.
C synopsis simInt simStopSimulation()
C parameters
None
C return value
-1 in case of an error, 0 if the operation could not be performed. >0 in case of success.
Lua synopsis number result=sim.stopSimulation()
Lua parameters
Same as C-function
Lua return values
Same as C-function
Remote API equiv.
B0-based remote API: simxStopSimulation
Legacy remote API: simxStopSimulation
