Regular API function

simSetStringNamedParam / sim.setStringNamedParam

Description Sets a named string or buffer value. See also sim.getStringNamedParam.
C synopsis simInt simSetStringNamedParam(const simChar* paramName,const simChar* stringParam,simInt paramLength)
C parameters
paramName: the key or name of the parameter
stringParam: the string or buffer value
paramLength: the size of the string or buffer value
C return value
-1 if operation was not successful. 0 if an existing parameter was replaced, 1 if the parameter was set
Lua synopsis number result=sim.setStringNamedParam(string paramName,string stringParam)
Lua parameters
Same as C-function
Lua return values
Same as C-function
