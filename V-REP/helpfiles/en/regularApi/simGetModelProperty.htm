Regular API function

simGetModelProperty / sim.getModelProperty

Description Retrieves the properties of a model. See also sim.setModelProperty, sim.getObjectProperty and sim.getObjectSpecialProperty.
C synopsis simInt simGetModelProperty(simInt objectHandle)
C parameters
objectHandle: handle of the object that serves as the model base
C return value
model property values, or -1 if operation was not successful
Lua synopsis number property=sim.getModelProperty(number objectHandle)
Lua parameters
Same as C-function
Lua return values
Same as C-function. See also the boolean operators in Lua.
Remote API equiv.
B0-based remote API: -
Legacy remote API: simxGetModelProperty
