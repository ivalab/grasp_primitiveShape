Calculation module properties dialog

The calculation module properties dialog is located at [Menu bar --> Tools --> Calculation module properties]. You can also open the dialog with a click on its toolbar button:

[Calculation module properties toolbar button]


The calculation module properties dialog displays properties related to calculation modules. The dialog is divided into 4 parts:

  • Collision detection: properties related to collision detection.
  • Distance calculation: properties related to minimum distance calculation.
  • Inverse kinematics: properties related to inverse kinematics.
  • Dynamics: properties related to dynamics.

    • 4 buttons in the upper part of the dialog allow selecting the desired type of properties to display:
