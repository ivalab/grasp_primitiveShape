Movement along a Path

The current intrinsic path position is indicated as a red sphere on the path (if the path position display is enabled). The position of the sphere along the path can be controlled at any time during simulation with sim.setPathPosition. Make sure you understand how path positions or the path length is calculated.

To actually make an object move along the path, you will first need to attach the object to a dummy, then assign the dummy to follow the path position (with an optional offset). Refer to the section on dummies for more details.

Alternatively, you can have an object follow a path by using the sim.followPath, or sim.getPositionOnPath, sim.setObjectPosition, sim.getOrientationOnPath and sim.setObjectOrientation command.
