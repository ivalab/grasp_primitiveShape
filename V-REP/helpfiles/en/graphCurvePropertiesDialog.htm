XY graph and 3D curve dialog

The XY graph and 3D curve dialog is part of the graph properties. The dialog displays the settings and parameters related to XY graphs or 3D curves of the last selected graph object.

[3D curve dialog]


  • Add new curve: allows adding a new XY graph or 3D curve item. The new item will be displayed in the dialog's list, and its name can be edited. Clicking the Add new curve button opens following dialog:

    • [3D curve selection dialog]

  • X- / Y- / Z-value = : allows specifying a data stream for each one of the 3 (or 2 in case of an XY graph) curve dimensions. Only data streams that are being listed in the data stream recording list can be selected. Refer to the graph properties for more details.

  • Curve is visible: determines whether the selected curve will be visible.
  • Link points: determines whether individual (x,y) or (x,y,z) values of the selected curve will be linked with a line.
  • Display label: determines whether the selected curve's label will be visible.
  • Adjust color: allows adjusting the color of the selected curve.
  • Duplicate to static: when a curve is selected, clicking this button will duplicate the curve as a static curve. Static streams or static curves (displayed as stippled lines) are not modified or cleared when a new simulation is started and are used to compare results from one simulation run to another. Data streams or curves can also be made static by selecting them in a graph view (except for floating views) then clicking [Popup menu --> Add --> Static duplicate of curve]. Static streams or curves can be cleared with the remove all static streams/curves button in the graph properties dialog.
  • Curve always on top: determines whether the selected curve will always be visible, even if hidden by other objects. This property is only available for 3D curves.
  • Relative to graph: if selected, then the selected curve will be displayed relative to the graph object's reference frame. This property is only available for 3D curves.
  • Relative to world: if selected, then the selected curve will be displayed relative to the absolute reference frame. This property is only available for 3D curves.
  • Curve width: the width of the selected curve. This property is only available for 3D curves.
