Mill volume dialog

The mill volume dialog is part of the mill properties. The dialog displays the cutting volume settings and parameters of the last selected mill. If no object is selected, the dialog is inactive. If more than one mill is selected, then some parameters can be copied from the last selected mill to the other selected mills (Apply to selection-buttons):

[Mill volume dialog]


  • Volume type: the type of the volume. Four types are supported: pyramid, cylinder, disc and cone. Following figures describe the parameters characterizing each detection volume:

    • [Pyramid and cylinder-type cutting volume parameters]


    [Disk and cone-type cutting volume parameters]


  • Face number, face number far and subdivisions far can be used to describe the cutting volume with more or less precision. More precision also implies longer calculation times in general, however when continuous milling is required (opposed to spot-cutting), a smoother volume could perform faster and with better results.
