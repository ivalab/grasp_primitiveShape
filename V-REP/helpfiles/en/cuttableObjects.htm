Cuttable objects

Cuttable objects are objects that can be cut by mills. For now, only simple non-pure shapes are cuttable, but a future V-REP release will also include cuttable volumes.

Collections are also cuttable, since they might contain cuttable objects.

Cuttable objects can have their cuttable property individually enabled or disabled (disabled by default). This can be set in the object common properties or through the sim.setObjectSpecialProperty API function.

Additionally, cuttable objects can have their cuttable property overridden depending on their related model properties (if they are part of a model). Refer to the model dialog for more information.
