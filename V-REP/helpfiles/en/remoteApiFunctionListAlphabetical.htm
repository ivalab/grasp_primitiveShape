Remote API function list

The remote API, or legacy remote API, should not be mixed-up with the BØ-based remote API, which is a newer version of the remote API that is more flexible, easier to use and most importantly, much simpler to extend.

If the function you are looking for is not listed below, then make sure to have a look at the generic function simxCallScriptFunction, which allows you to implement virtually any remote API function.
