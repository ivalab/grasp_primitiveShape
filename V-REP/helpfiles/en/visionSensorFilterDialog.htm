Vision sensor filter dialog

The vision sensor filter dialog is part of the vision sensor properties. The dialog displays the filter components to be applied to an acquired image. If no object is selected, the dialog is inactive. If more than one vision sensor is selected, then some parameters can be copied from the last selected vision sensor to the other selected vision sensors (Apply to selection-buttons):

[Vision sensor filter dialog]


  • Add filter: allows appending the selected filter component to the filter component list below. Components in the list can then be ordered as desired with the up/down buttons. Some components can be adjusted with a double-click.
  • Filter Component is enabled: allows enabling/disabling the selected filter component in the list.
